## 📌 About the Project  
AIROGS (Artificial Intelligence for Robust Glaucoma Screening) is a deep learning-based system designed to detect glaucoma from retinal fundus images.  

This project focuses on building a robust AI model that works effectively in real-world clinical conditions, including handling low-quality and ungradable images.  

## 🎯 Objectives  
- Detect glaucoma automatically using AI  
- Improve early diagnosis and reduce blindness  
- Handle real-world image variations  
- Identify Ungradable images  
- Provide confidence scores for predictions  

## ⚙️ Features  
- Fundus Image Upload  
- Image preprocessing  
- CNN Model (MobileNetV2)  
- Classification: Glaucoma / Normal / Ungradable  
- Confidence Score  
- PDF Report Generation  
- Dashboard  
- Secure Authentication  

## 🏗️ System Architecture  
Data Acquisition → Preprocessing → Feature Extraction → AI Model → Prediction → Output
